telegram bot on python. <\br>
Имя бота: @TheDoTheThingBot
https://t.me/TheDoTheThingBot.
Проект представляет собой телеграм-бота, который по команде пользователя выдаёт расписание уроков группы 307
Чтобы поменять расписание на другое можно изменить содержимое словаря schedule в файле data.py
Чтобы воспользоваться ботом нужно положить запустить файл thing.py в терминале и написать боту /start
Проект допускает расширения, например динамическое изменение расписания админами и модераторами и назначение модераторов админами
